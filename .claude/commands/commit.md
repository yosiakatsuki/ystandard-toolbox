# /project:commit

現在の更新内容をコミットします。

## 動作

1. `git status` と `git diff` で変更内容を確認
2. 変更内容に基づいて日本語で簡潔なコミットメッセージを作成
3. 詳細メッセージがある場合は末尾に `@Claude` を追記
4. コミットを実行

## 要件

-   コミットメッセージは日本語で簡潔に作成
-   変更内容を的確に表現する
-   「Generated with Claude Code」などの長い署名は使用しない
-   末尾には `@Claude` のみ追記

## 使用例

```
/project:commit
```
