# [開発者向け] カスタムブロックスタイル追加・読み込み機能

フック : `ystdtb_custom_block_styles`

パラメーター:
| パラメーター名 | 内容 |
| -------------- | ---- |
| block_name | ブロック名 (例: `core/paragraph`) |
| name | ブロックスタイル名 (例: `fancy-style`) |
| label | ブロックスタイルのラベル (例: `Fancy Style`) |
| url | ブロックスタイルのCSSファイルURL (例: `https://example.com/path/to/style.css`) |
| var | ブロクスタイルCSSのバージョン文字列 |
| editor_style | エディタースタイルCSSのURL (例: `https://example.com/path/to/editor-style.css`) |
| editor_var | エディタースタイルCSSのバージョン文字列 |
