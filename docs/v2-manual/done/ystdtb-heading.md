# 見出しデザイン編集機能

## 対象 URL

<https://wp-ystandard.com/manual/ystdtb-heading/>

## 対象機能

yStandard Toolbox の「見出しデザイン編集」機能。
管理画面から見出し用のデザインスタイルを作成し、見出しレベルへの割り当てや、ブロックスタイルとしての利用を設定できる。

## v2 での変更点

- 設定画面を React ベースの UI に刷新
- 「スタイル編集」と「割り当て設定」を分離
- 見出しデザインを複数のスタイルとして管理可能
- 作成したスタイルを見出しブロックのブロックスタイルとして利用可能
- 作成したスタイルを段落ブロックのブロックスタイルとして利用可能
- yStandard 以外のテーマでも、ブロックスタイルとして利用しやすい構成に変更
- レスポンシブ設定のデータ構造を統一

## 既存マニュアルから更新する内容

- 旧 UI のスクリーンショットを v2 UI に差し替える
- 「見出しレベルを選んで編集する」説明から、「スタイルを作成・編集して割り当てる」説明へ変更する
- ブロックスタイルとして使う設定を追加する
- 段落ブロックにも適用できることを追加する
- yStandard 以外のテーマでは、割り当て設定が反映されない場合があることを追記する

## 一押しポイント

見出しデザイン編集機能を使うと、サイト内で繰り返し使う見出しデザインを管理画面から作成できます。
v2 では、ページ全体の h2・h3 などにまとめて割り当てるだけでなく、必要な場所だけに使える「ブロックスタイル」としても登録できるようになりました。

見出しブロックだけでなく段落ブロックにも同じデザインを適用できるため、目立たせたいリード文や補足テキストなどにも、サイトの雰囲気に合った装飾を使えます。

## 本文案

### 見出しデザイン編集機能でできること

見出しデザイン編集機能では、投稿本文内の h1〜h6、投稿タイトル、固定ページタイトル、一覧ページタイトルなどに使う見出しデザインを作成できます。

作成した見出しデザインは、次の 2 つの方法で利用できます。

- 見出しレベルへ割り当てる
- ブロックスタイルとして使う

「見出しレベルへ割り当てる」を使うと、投稿本文内の h2 や h3 など、同じ見出しレベルへまとめてデザインを反映できます。

「ブロックスタイルとして使う」を使うと、ブロックエディター上で必要なブロックだけにデザインを適用できます。
見出しブロックのほか、段落ブロックにもスタイルを登録できます。

### 設定画面を開く

WordPress 管理画面から「[ys] Toolbox」→「見出しデザイン編集」を開きます。

v2 の見出しデザイン編集画面には、次の 2 つの編集モードがあります。

- スタイル編集
- 割り当て設定

まず「スタイル編集」で見出しデザインを作成し、必要に応じて「割り当て設定」で h2 や h3 などへ割り当てます。

### スタイルを追加する

「スタイル編集」を開き、「スタイル追加」をクリックします。

スタイル追加画面では、次の項目を入力します。

- スタイルID
- スタイル名

スタイルIDは、英数字とハイフンのみ使用できます。最初の文字は英字にしてください。
ハイフンは1回のみ使用できます。

スタイル名は、管理画面やブロックエディター上に表示される名前です。
編集画面などで表示された時に分かりやすい名前を付けてください。

### スタイルを編集する

編集するスタイルを選択すると、左側に設定パネル、右側にプレビューが表示されます。
プレビューを確認しながら、見出しの見た目を調整します。

主な設定パネルは次のとおりです。

- スタイル名
- デザインテンプレート
- ブロックスタイル
- 文字設定
- 背景設定
- 枠線設定
- 余白設定
- 幅設定
- 高さ設定
- 上級者向け
- `::before` 設定
- `::after` 設定

設定を変更したら、「スタイル設定を更新」をクリックして保存します。
保存する前に別のスタイルへ切り替えたり、編集モードを切り替えたりすると、編集中の内容は破棄されます。

### デザインテンプレートを使う

「デザインテンプレート」から、あらかじめ用意されたデザインを選択できます。
テンプレートを選んだあとに、文字色・背景色・余白・線などを調整して、サイトに合わせたデザインに仕上げます。

最初から細かく設定するのが難しい場合は、テンプレートを選んでから必要な部分だけ変更する方法がおすすめです。

### ブロックスタイルとして使う

「ブロックスタイル」パネルでは、作成したスタイルをブロックエディターで選択できるようにするかを設定します。

設定できる項目は次の 2 つです。

- 見出しブロックのスタイルとして使う
- 段落ブロックのスタイルとして使う

「見出しブロックのスタイルとして使う」を ON にすると、見出しブロックのスタイル一覧にこのデザインが追加されます。

「段落ブロックのスタイルとして使う」を ON にすると、段落ブロックのスタイル一覧にこのデザインが追加されます。
見出し風のリード文や、強調したい短い文章を作るときに便利です。

### 見出しレベルへ割り当てる

「割り当て設定」では、作成したスタイルを見出しレベルやページタイトルへ割り当てます。

割り当てできる主な項目は次のとおりです。

- h1
- h2
- h3
- h4
- h5
- h6
- 投稿タイトル
- 固定ページタイトル
- 一覧ページタイトル

yStandard シリーズのテーマを利用している場合は、次の項目も表示されます。

- サイドバーウィジェット
- フッターウィジェット

各項目のセレクトボックスから適用したいスタイルを選び、「割り当て設定を更新」をクリックします。

割り当てを外したい場合は、「-- 割り当てなし --」を選択して保存します。

### yStandard 以外のテーマで使う場合

yStandard 以外のテーマでは、テーマの HTML 構造によって、投稿タイトル・一覧ページタイトルなどへの割り当て設定が反映されない場合があります。

割り当て設定が反映されない場合は、ブロックスタイルとして登録し、ブロックエディター上で必要なブロックへ適用してください。

### スタイルを削除する

不要になったスタイルは、「スタイル編集」で対象のスタイルを選択し、「スタイルを削除」をクリックします。

削除したスタイルを見出しレベルに割り当てていた場合は、表示を確認し、必要に応じて別のスタイルを割り当て直してください。
