# v1 からの見出し設定の移行方法

## 対象 URL

未定。
既存の見出しデザイン編集機能ページから導線を追加する想定。

## 対象機能

yStandard Toolbox v1 系の見出し設定を、v2 の見出しデザイン編集機能へ移行する手順。

## v2 での変更点

v2 では、見出し設定の保存形式と編集画面が変更されています。
v1 の見出し設定が残っている場合、見出しデザイン機能は互換モードで動作します。

互換モードでは旧設定を使って表示を維持しますが、新しい見出しデザイン編集画面で設定を変更するには、v2 形式へ変換する必要があります。

## 既存マニュアルから更新する内容

- v1 設定がある場合に互換モードで動作することを説明する
- v2 の設定変換画面の操作手順を追加する
- 変換後に表示確認と調整が必要なことを明記する
- 旧設定で有効だった見出しだけが割り当て設定へ自動反映されることを説明する

## 本文案

### 移行が必要なケース

yStandard Toolbox v1 系で見出しデザイン編集機能を使っていたサイトでは、v2 へのアップデート後も旧設定をもとに見出しデザインが表示されます。

この状態では、見出しデザイン機能は互換モードで動作しています。
管理画面には、次のような案内が表示されます。

> [yStandard Toolbox]見出しデザイン機能は現在互換モードで動作しています。[ys]Toolbox → 見出しデザイン編集 メニューから新方式へ変換してください。

v2 の新しい編集画面で見出しデザインを編集するには、旧設定を新設定へ変換してください。

### 移行前に確認すること

設定変換を実行する前に、現在の見出しデザインが表示されているページを確認しておくことをおすすめします。

特に次のページは、変換後に見た目を比較しやすいように確認しておいてください。

- 投稿ページ
- 固定ページ
- 一覧ページ
- 見出しを多く使っている記事
- サイドバーやフッターにウィジェット見出しがあるページ

可能であれば、作業前にデータベースのバックアップを取ってください。

### 設定変換画面を開く

WordPress 管理画面から「[ys] Toolbox」→「見出しデザイン編集」を開きます。

旧設定が残っている場合は、通常のスタイル編集画面ではなく「旧設定から新設定への移行」画面が表示されます。

画面には次の案内が表示されます。

- 見出し設定は旧バージョンで動作していること
- 設定を変更するには新バージョンへ設定を変換する必要があること
- 変換後に表示の確認・調整が必要なこと

### 設定変換を実行する

「設定変換を実行する」をクリックします。

変換が完了すると、次の案内が表示されます。

- 設定移行が完了しました。
- ページ再読み込みをしてください。

案内に従って、ページを再読み込みしてください。

### 変換後に作成されるスタイル

変換後は、v1 の各見出し設定が v2 のスタイルとして作成されます。

作成されるスタイル名は、次のような形式になります。

- `v1:h1`
- `v1:h2`
- `v1:h3`
- `v1:h4`
- `v1:h5`
- `v1:h6`
- `v1:投稿タイトル`
- `v1:固定ページタイトル`
- `v1:一覧ページタイトル`
- `v1:サイドバー`
- `v1:フッター`

v1 で「使う」設定になっていた見出しは、v2 の「割り当て設定」に自動で割り当てられます。
v1 で使用していなかった見出しは、スタイルが作成されても自動割り当てされない場合があります。

### 変換後に表示を確認する

設定変換が完了したら、サイトの表示を確認します。

特に次の点を確認してください。

- 投稿本文内の h2〜h6 の見た目
- 投稿タイトルの見た目
- 固定ページタイトルの見た目
- 一覧ページタイトルの見た目
- サイドバーウィジェットの見出し
- フッターウィジェットの見出し
- `::before` / `::after` を使った装飾
- アイコンを使った装飾
- 背景画像を使った装飾
- スマートフォン・タブレット・デスクトップでの文字サイズや余白

変換後の見た目が完全に同じにならない場合があります。
必要に応じて「スタイル編集」で各スタイルを調整してください。

### 割り当てを確認する

「見出しデザイン編集」画面で「割り当て設定」を開きます。

h1〜h6、投稿タイトル、固定ページタイトル、一覧ページタイトルなどに、意図したスタイルが割り当てられているか確認します。

割り当てを変更したい場合は、各項目のセレクトボックスから別のスタイルを選択し、「割り当て設定を更新」をクリックしてください。

### ブロックスタイルとして使う場合

変換されたスタイルは、必要に応じてブロックスタイルとしても利用できます。

「スタイル編集」で対象スタイルを選択し、「ブロックスタイル」パネルを開いて、次の設定を ON にします。

- 見出しブロックのスタイルとして使う
- 段落ブロックのスタイルとして使う

ON にしたスタイルは、ブロックエディターのスタイル一覧から選択できるようになります。

### 変換に失敗した場合

設定の変換に失敗した場合、v1 の設定は削除されません。
再度「設定変換を実行する」を試してください。

変換後に表示が大きく崩れる場合は、バックアップから復元するか、変換後の `v1:` から始まるスタイルを手動で調整してください。

## 動作確認項目

- v1 設定がある状態で互換モードになる
- 見出しデザイン編集画面で移行画面が表示される
- 「設定変換を実行する」で v2 設定が作成される
- 変換完了後にページ再読み込み案内が表示される
- v1 設定の各見出しが `v1:` から始まるスタイルへ変換される
- v1 で有効だった見出しが割り当て設定へ反映される
- 変換後に通常の「スタイル編集」「割り当て設定」画面が表示される
- 投稿本文、投稿タイトル、固定ページタイトル、一覧ページタイトルの表示を確認する
- `::before` / `::after`、アイコン、背景、余白、線の見た目を確認する
