設定一覧

画面サイズによる非表示

既存のブロックに「モバイル・タブレット・PC のどれで非表示にするか」を設定できます。
※ブロックによっては設定が表示されない場合もあります。（ここでは段落ブロックと Box ブロックで確認します。）

モバイルのみ非表示

スマホでは見えず、タブレットと PC で表示されるパターンです。

段落ブロック

[段落] PC とタブレットで表示されます（モバイル非表示）

Box ブロック

[Box] PC とタブレットで表示されます（モバイル非表示）

タブレットのみ非表示

タブレットだけ見えず、スマホと PC で表示されるパターンです。

段落ブロック

[段落] PC とモバイルで表示されます（タブレット非表示）

Box ブロック

[Box] PC とモバイルで表示されます（タブレット非表示）

PC のみ非表示

PC では見えず、スマホとタブレットで表示されるパターンです。

段落ブロック

[段落] モバイルとタブレットで表示されます（PC 非表示）

Box ブロック

[Box] モバイルとタブレットで表示されます（PC 非表示）

モバイル + タブレット（PC のみ表示）

PC だけ表示したいときに使います。スマホ・タブレットの両方で非表示になります。

段落ブロック

[段落] PC のみで表示されます（モバイル + タブレット非表示）

Box ブロック

[Box] PC のみで表示されます（モバイル + タブレット非表示）

タブレット + PC（モバイルのみ表示）

スマホだけ表示したいときに使います。タブレット・PC の両方で非表示になります。

段落ブロック

[段落] モバイルのみで表示されます（タブレット + PC 非表示）

Box ブロック

[Box] モバイルのみで表示されます（タブレット + PC 非表示）

モバイル + PC（タブレットのみ表示）

タブレットだけ表示したいときに使います。スマホ・PC の両方で非表示になります。

段落ブロック

[段落] タブレットのみで表示されます（モバイル + PC 非表示）

Box ブロック

[Box] タブレットのみで表示されます（モバイル + PC 非表示）
