設定の組み合わせ例

よく使われる設定の組み合わせをいくつかご紹介します。そのまま使ったり、アレンジのヒントにしてみてください。

記事下の基本シェアボタン

記事を読み終えたあとに置く、いちばんオーソドックスな構成です。デフォルトの円ボタンで X・Facebook・はてなブックマーク・LINE をまとめて用意しています。

サイドバーにすっきり置きたい

サイドバーや狭いスペースでも場所を取らないよう、アイコンデザイン・左寄せ・SNS 控えめの組み合わせにしています。

X を盛り上げたいブロガー向け

X でのシェアを中心にしたい場合の構成です。via アカウントとハッシュタグを設定しておくと、ツイートされたときに自分のアカウントとタグが自動で付きます。

公式ボタンで信頼感を演出

各 SNS が公式で提供しているボタンを使う構成です。見慣れたデザインなのでクリックされやすく、信頼感を出したいときにおすすめです。公式ボタンは Bluesky に対応していないため OFF にしています。

上下両方設定

上下のテキストを両方設定した例です。

Twitter 用オプション

X ボタン・Twitter ボタンからシェアしたときの URL パラメータに反映される設定です。シェアしてくれた読者のツイートに、自分のアカウントやハッシュタグを自動で付けられます。

via アカウント

ツイート本文の末尾に「via @アカウント名」として付与されます。「@」を除いたユーザー名だけを入力してください。

未設定（デフォルト）
設定あり（例: ystandard）

おすすめアカウント

ツイート送信後に、おすすめフォローとして表示されるアカウントです。関連サービスや運営アカウントなどを指定します。「@」を除いたユーザー名だけを入力してください。

未設定（デフォルト）
設定あり（例: ystandard）

ハッシュタグ

ツイート本文に追加されるハッシュタグです。「#」を除いて、カンマ区切りで複数指定できます。

未設定（デフォルト）
設定あり（例: ystandard, wordpress）
