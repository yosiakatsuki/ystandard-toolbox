設定一覧

タイプ

ボックスのラベル位置や表示形式を選べます。ラベル無し・外側・内側・全幅・線上の 5 種類から選択できます。

ラベル無し（label-none・デフォルト）

ラベル無しはシンプルに枠だけ表示します。

外側ラベル（label-out）

ポイント

外側ラベルは枠の上に飛び出した見た目です。

内側ラベル（label-in）

ポイント

内側ラベルは枠の内側上部に配置されます。

全幅ラベル（label-wide）

お知らせ

全幅ラベルは枠と同じ幅で表示されます。

線上ラベル（label-line）

補足

線上ラベルは下線に乗せたラベル風の見た目です。

ボックス設定

ボックスの背景色・文字色・枠線・角丸・余白を調整できます。

背景色（プリセット）

プリセット色 ys-blue を背景に設定しています。

背景色（カスタム）

カスタム色で背景を塗っています。

文字色（カスタム）

文字色をカスタム色にしています。

枠線色（プリセット）

プリセット色 ys-red を枠線色に指定しています。

枠線の太さ（3px）

枠線の太さを 3px にしています。

枠線のスタイル（dotted）

枠線のスタイルを点線にしています。

角丸（8px）

角丸 8px にしています。

内側の余白（一括 16px）

内側余白を一括 16px にしています。

内側の余白（レスポンシブ）

余白をデバイスごとに変えるレスポンシブ設定です（デスクトップ 32px / タブレット 24px / モバイル 16px）。

内側の余白（プリセット）

余白にプリセット値（spacing/30）を指定しています。

線上ラベル（label-line）＋ レスポンシブ余白

補足

線上ラベル + レスポンシブ余白（デスクトップ 32px / タブレット 24px / モバイル 16px）。画面幅で padding-top とラベルの横位置が tablet/mobile 値に追従します。
