スライダーブロック 設定バリエーション例

スライダーブロックの各設定パターンを網羅した examples ページです。組み合わせ例・子ブロックの種類・各設定一覧の順に並べています。

設定の組み合わせ例

実用的な設定パターンの例です。サイトのデザインに合わせてカスタマイズしてください。

基本（デフォルト挙動）

デフォルト設定のまま使う例です。自動再生・8 秒間隔・ループ ON。

ヒーロースライダー（全幅 + 高さ500px + ナビ + ドット）

ページ最上部のメインビジュアルとして使う想定の設定です。

商品カードスライダー（複数表示 + デバイス別）

モバイル 1 列 / タブレット 2 列 / デスクトップ 3 列で表示する設定です。

フェードスライダー（fade + delay 5 秒）

スライド切り替え時にフェードアニメーションを使う設定です。

アスペクト比 16:9 + ナビゲーション色カスタム

高さを 16:9 のアスペクト比で固定 + ナビゲーション矢印の色をオレンジに指定する設定です。
