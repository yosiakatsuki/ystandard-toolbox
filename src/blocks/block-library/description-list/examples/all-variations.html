設定の組み合わせ例

よく使われる組み合わせをまとめました。そのまま使ったり、アレンジのヒントにしてみてください。

用語集

いちばんシンプルな「用語 + 説明」の繰り返しです。専門用語の解説ページや記事末尾の補足情報にもぴったりです。

WordPress
ブログや Web サイトを作るためのオープンソース CMS です。世界中のサイトで使われています。
Gutenberg
WordPress のブロックエディターの名前です。記事をブロック単位で組み立てられます。
yStandard
シンプルで使いやすさを追求した無料 WordPress テーマです。

イベント日程お知らせ

横並びレイアウトで「開催日」や「内容」を並べる、イベント日程のお知らせなどにぴったりの構成です。dt の幅を固定してボーダーで区切ると表のように見せられます。

開催日

20xx年4月12日

場所

◯◯市民会館 A会議室

料金

無料

スペック表

商品スペックや仕様一覧のように、モバイルでも横並びを維持したい場面向けの構成です。dt を細めに固定するとコンパクトに並びます。

サイズ

重量

約 520 g

材質

行間を調整できます。長い用語で改行されたときの読みやすさを整えたい場合に使います。

横並びレイアウト（dl-column）

dt と dd を横並びに表示するブロックです。dl の中に挿入して使います。モバイル・タブレットで縦積みに切り替える設定、dt の幅指定、ボーダーなどが設定できます。dl-column の中に入れられるのは dt と dd-box のみです（dd-simple は非対応）。
