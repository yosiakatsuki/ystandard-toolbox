記事一覧ブロックの設定例・使用例

記事一覧ブロックで設定できる各オプションの表示例をまとめたページです。設定の組み合わせによる表示の違いを確認できます。

設定の組み合わせ例

複数の設定を組み合わせた実用的なパターンです。サイトのデザインに合わせてカスタマイズしてください。

ブログ記事一覧（2列 + 概要つき）

記事の内容がわかりやすい、概要文つきの2列カード表示です。

ギャラリー風（4列 + 正方形 + メタなし）

画像を主役にした、ギャラリーのような表示です。メタ情報を非表示にすることで、よりビジュアルに特化した一覧になります。

ニュース一覧（リスト + 画像なし + 概要つき）

お知らせやニュースのようなテキスト重視の一覧に適しています。

おすすめ記事（リスト + 正方形画像 + 概要つき）

正方形の画像と概要文で記事の内容をしっかり伝えるレイアウトです。

サイドバー向け（シンプル + 10件 + 日付のみ）

サイドバーウィジェットやフッターなど狭いスペースに最適な、コンパクトな一覧です。

最近更新された記事（3列 + 概要つき + 更新日順）

リライトした記事を目立たせたいときに便利な、更新日順の表示です。

ピックアップ（ランダム + オフセット）

最新記事を除いた中からランダムに表示する設定例です。「こんな記事もあります」のような導線づくりに活用できます。

基本設定

表示する記事の件数や並び順を設定します。モバイル端末で閲覧したときの表示件数を別途指定することもできます。

表示件数の設定例

表示件数を変更することで、一覧に表示される記事の数を調整できます。

3件（デフォルト）
1件のみ表示
6件表示
12件表示
PC・モバイルで件数を変える（PC:6件 / モバイル:2件）

モバイル端末ではスクロール量を減らすため、表示件数を少なくする設定例です。

並び順の設定例

記事の表示順を「公開日」「更新日」「タイトル」「ランダム」から選べます。それぞれ昇順・降順の指定が可能です。

公開日が新しい順（デフォルト）
公開日が古い順
最近更新された順
タイトルのあいうえお順
ランダム表示

ページを読み込むたびに表示される記事が変わります。

表示設定

記事一覧のデザインタイプとカラム数（列数）を設定します。デザインタイプは「カード」「リスト」「シンプル」の3種類から選べます。

デザインタイプの比較

カード（デフォルト）

画像を上部に大きく表示するカード型レイアウトです。ビジュアルを重視した一覧に向いています。

リスト

画像を左、テキストを右に配置する横並びレイアウトです。コンパクトに多くの記事を表示できます。

シンプル

画像なしのテキストのみの一覧です。サイドバーやフッターなど狭いスペースでの使用に適しています。

PC・モバイルでデザインを変える（PC:カード / モバイル:リスト）

PCではカード表示、モバイルではコンパクトなリスト表示に切り替える設定例です。

PC・モバイルでデザインを変える（PC:カード / モバイル:シンプル）

カラム数（列数）の設定例

PC・タブレット・モバイルそれぞれの画面幅に応じたカラム数を設定できます。カード表示との組み合わせで使用します。

3列（デフォルト: PC:3 / タブレット:3 / モバイル:1）
2列（PC:2 / タブレット:2 / モバイル:1）
4列（PC:4 / タブレット:2 / モバイル:2）

PC画面では4列で表示し、タブレット以下では2列に切り替える設定例です。

1列（カードを縦に並べる）

サムネイル設定

記事のアイキャッチ画像に関する設定です。画像の表示・非表示、アスペクト比（縦横比）、画像サイズを変更できます。

画像の表示・非表示

画像あり（デフォルト）
画像なし

テキストのみのカード表示になります。画像を使わずにコンパクトな一覧にしたい場合に便利です。

アスペクト比（縦横比）の設定例

画像のアスペクト比を変更することで、画像の表示領域の形を調整できます。画像自体は切り抜かれて表示されます。

16:9（デフォルト・横長ワイド）
4:3（横長スタンダード）
3:2
1:1（正方形）
2:1（横長パノラマ）
3:1（横長バナー）
9:16（縦長）
4:5（縦長・SNS風）
2:3（縦長ポートレート）
PC・モバイルでアスペクト比を変える（PC:16:9 / モバイル:1:1）

PC画面ではワイドな画像、モバイルでは正方形に切り替える設定例です。

画像サイズの設定例

WordPressに登録されている画像サイズを選択できます。表示サイズに合った画像を使うことで、ページの読み込み速度が改善されます。

フルサイズ（デフォルト）
サムネイル（150×150）
中サイズ
大サイズ

リスト表示でのアスペクト比

リスト表示ではサムネイルが小さく表示されるため、正方形（1:1）や4:3との組み合わせがおすすめです。

リスト + 正方形（1:1）
リスト + 4:3

メタ情報の表示設定

投稿日やカテゴリー、概要文の表示・非表示を切り替えられます。

日付・カテゴリーの表示パターン

日付・カテゴリーともに表示（デフォルト）
日付のみ表示
カテゴリーのみ表示
日付・カテゴリーともに非表示

タイトルのみのすっきりした表示になります。

概要文の設定例

記事の抜粋（概要文）を表示できます。表示行数を指定すると、長い概要文は自動的に省略されます。

概要なし（デフォルト）
概要あり（2行まで表示）
概要あり（1行のみ表示）
概要あり（4行まで表示）

デザインタイプ別のメタ情報表示

デザインタイプごとにメタ情報の見え方が変わります。実際に使用するデザインで確認してください。

リスト表示 + 全メタ情報 + 概要2行
シンプル表示 + 日付のみ
シンプル表示 + メタ情報なし(タイトルのみ)
シンプル表示 + 概要あり

絞り込み設定

表示する記事を投稿タイプやカテゴリー・タグで絞り込むことができます。

投稿タイプの切り替え

デフォルトでは「投稿」が選択されていますが、「固定ページ」やカスタム投稿タイプに切り替えることもできます。

投稿（デフォルト）
固定ページ

カテゴリー・タグでの絞り込み

投稿タイプを選択した後、分類（タクソノミー）とターム（カテゴリー名やタグ名）を指定して絞り込みます。以下は「カテゴリー」で「未分類」を指定した例です。サイトに合わせてタームを変更してください。

カテゴリー「未分類」で絞り込み

高度な絞り込み設定

投稿IDやスラッグ（投稿名）を直接指定して、特定の記事だけを表示できます。この設定を使うと、絞り込み設定（カテゴリー・タグ指定）はクリアされます。

投稿IDで指定

表示したい記事の投稿IDをカンマ区切りで入力します。投稿IDは記事の編集画面URLなどから確認できます。

投稿ID 1, 2, 3 を指定

投稿名(スラッグ)で指定

表示したい記事のスラッグ（URLの一部）をカンマ区切りで入力します。

スラッグ「hello-world」を指定

親ページで絞り込み

固定ページなど階層型の投稿タイプで、特定の親ページに属する子ページだけを表示できます。

親ページID 2 の子ページを表示

高度な表示設定

表示開始位置（オフセット）を指定することで、先頭の記事をスキップして表示できます。たとえば他のブロックで最新記事を表示している場合に、重複を避けるために使えます。

オフセット（表示開始位置）の設定例

オフセットなし（デフォルト・1件目から表示）
オフセット3（4件目から表示）

最新3件を他のブロックで表示している場合に、4件目以降を表示する設定例です。

PC・モバイルでオフセットを変える（PC:3 / モバイル:1）
