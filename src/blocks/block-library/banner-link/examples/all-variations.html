設定の組み合わせ例

よく使われる設定の組み合わせをいくつかご紹介します。そのまま使ったり、アレンジのヒントにしてみてください。

シンプルなバナーリンク

画像を使わず、背景色とテキストだけで仕上げたシンプルな構成です。さっと作りたいときにおすすめですよ。

お知らせページへ
最新情報をチェックする

写真バナー

背景に写真を置いて、その上にテキストを重ねるスタイルです。暗めのオーバーレイをかけると文字が読みやすくなります。

ギャラリーを見る
これまでの制作実績をご紹介しています

カード風バナー

角丸・影・最大幅を組み合わせると、カードのような見た目になります。サイドバーや2カラムレイアウトにも合いますよ。

詳しくはこちら
サービスの詳細ページへ移動します

キャンペーン告知バナー

グラデーション背景にテキストサイズを大きめに設定すると、目を引くバナーに仕上がります。キャンペーンやイベント告知にぴったりです。

バナー設定

バナーの見た目を決める設定です。背景画像や背景色、サイズなどをお好みに合わせて調整できます。

画像設定

バナーの背景に画像を設定できます。画像を設定すると焦点（フォーカスポイント）も調整できるようになりますよ。

画像なし（デフォルト）

焦点を左上寄りに設定した例です。被写体の位置に合わせて調整すると、画像がきれいに見えます。

焦点を変更
左上寄りに焦点を設定しています
